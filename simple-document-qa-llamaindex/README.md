# Simple document Q&A with LlamaIndex

Just following the [Llama index starter](https://docs.llamaindex.ai/en/stable/getting_started/starter_example.html) here.

## Prerequisites

- a working Python environment with `python3-venv` installed
- an OpenAI API key

## How to run

- `cp .env.example .env` and fill in the blanks
- `pip install -r requirements.txt`
- talk to the bot with `python3 app.py question`
